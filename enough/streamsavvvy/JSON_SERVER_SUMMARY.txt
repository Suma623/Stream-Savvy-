PACKAGE.JSON SCRIPTS ADDED:
----------------------------
- npm run json-server   → Start JSON Server on port 3001
- npm run fetch-movies  → Fetch movies from TMDB and save to db.json


HOW TO USE:
-----------

1. START JSON SERVER:
   npm run json-server

2. TEST API (in new terminal):
   curl http://localhost:3001/movies

3. OPEN EXAMPLE FRONTEND:
   Open examples/frontend-example.html in browser

4. FETCH REAL MOVIES:
   npm run fetch-movies


AVAILABLE ENDPOINTS:
--------------------
GET    /movies       - Get all movies
GET    /movies/:id   - Get single movie
POST   /movies       - Add new movie
PUT    /movies/:id   - Update movie
PATCH  /movies/:id   - Partial update
DELETE /movies/:id   - Delete movie


MANUAL EDITING:
---------------
Just open db.json in your text editor, add/edit movies, and save.
JSON Server will automatically reload the data.


FRONTEND EXAMPLES:
------------------

1. Fetch API (see examples/frontend-example.html):
   const response = await fetch('http://localhost:3001/movies');
   const movies = await response.json();

2. React Component (see examples/MoviesList.jsx):
   import MoviesList from './examples/MoviesList';
   <MoviesList />

3. Axios Helper (see examples/axios-example.js):
   import moviesApi from './examples/axios-example';
   const movies = await moviesApi.getAllMovies();
